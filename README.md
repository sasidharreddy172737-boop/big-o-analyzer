# BIG (O) — Complexity Analyzer

Analyzes Python code and gives
Big O time and space complexity.

## Live Demo
https://sasidharreddy172737-boop.github.io/big-o-analyzer/


## Features
- Detects O(1) to O(n³)
- Color coded results
- Space complexity
- Built with HTML CSS JS

## How to Use
1. Paste Python code
2. Click Calculate
3. See Big O result
